## ğŸ‡²ğŸ‡¹ About Malta International Airport (LMML)

Malta International Airport (LMML) is the only airport serving the Maltese islands. Located between Luqa and Gudja on the island of Malta, it serves as the main hub for Air Malta and handles over 7 million passengers annually. This scenery provides an accurate, up-to-date representation based on the latest Google Maps satellite imagery and real-world airport charts.

## ğŸ“‹ Installation Instructions

### Manual Installation
1. Download the latest release from [GitHub Releases](https://github.com/vikingnope/LMML-MSFS/releases/latest)
2. Extract the downloaded ZIP file
3. Copy the `vikingstudios-airport-lmml-malta` folder to your MSFS Community folder:
   - **Steam**: `C:\Users\[Username]\AppData\Roaming\Microsoft Flight Simulator\Packages\Community\`
   - **Microsoft Store**: `C:\Users\[Username]\AppData\Local\Packages\Microsoft.FlightSimulator_*\LocalCache\Packages\Community\`

### GSX Profile Installation (Optional)
If you have GSX (Ground Services X) installed:
1. Locate the GSX profile file (if included in the download)
2. Copy the profile to your GSX airports folder:
   - **Default location**: `C:\Users\[Username]\AppData\Roaming\virtuali\GSX\MSFS\airports\`
3. Restart MSFS and GSX for the profile to take effect

### Alternative Download
Alternatively, you can download from [Flightsim.to](https://flightsim.to/file/75544/lmml-malta-international-airport-malta-early-access) and follow their installation guide.
